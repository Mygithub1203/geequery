# Chapter-20  附录二 数据库兼容性说明

完全支持以下数据库——

* Oracle 9.0以上版本、
* MySQL 5.0以上版本
* MariaDB 5.5测试通过。 10.x版本尚未测试。
* Postgresql 8.2以上版本
* Microsoft SQL Server 2003或以上
* HSQLDB 2.0以上版本
* Apache Derby 10.5以上版本
* SQL ite（部分DDL该DBMS无法支持）

部分支持——

* Gbase

计划支持——因时间或工作量因素，Dialect尚未编写的

* IBM DB2
* Sybase
* MongoDB -JDBC
